1.   Change Directory: cd directoryName OR cd .. to navigate up a directory
2.   Create a Directory: mkdir folderName
3.   Rename a Directory: mv originalFilename desiredFileName
4.   List the contents of a directory: ls
5.   Move a file to a new location: mv /originalPath /desiredPath
6.   Copy a file to a new location: cp /originalPath /desiredPath
7.   Search for a string in a file: grep someString filename
8.   Access the shell of a remote machine: ssh 
9.   Remove a directory folder: rmdir
